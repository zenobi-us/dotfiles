### [Decision] Content Structure

**Frontmatter:**
```yaml
---
title: { Decision Title }
projectId: { ProjectId }
status: { pending | decided | unresolved | superseded }
---
```

**Links:**
- to one or more [Research]
- to affected [Epic], [Story], [Task]
- if superseded: to replacement [Decision]

**Sections:**
- **Context**
- **Options Considered**
- **Decision**
- **Rationale**
- **Trade-offs**
- **Implications**
- **Review Schedule**
