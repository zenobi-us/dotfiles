# Artifact Status Transitions (Miniproject-Aligned)

## [Idea]
`triaged -> incubating -> promoted | rejected`

## [Epic]
`planning -> active -> completed`
Alternative exits: `on-hold`, `cancelled`

## [Story]
`draft -> approved -> in-progress -> in-review -> completed`
Alternative: `blocked`, `cancelled`
Completion gate: `test_coverage MUST be full`.

## [Task]
`todo -> in-progress -> in-review -> completed`
Alternative: `blocked`, `cancelled`

## [Research]
`in-progress -> complete`
Alternative: `inconclusive`, `superseded`

## [Decision]
`pending -> decided`
Alternative: `unresolved`, `superseded`

## [Learning]
`active -> archived`

## [Retrospective]
`in-progress -> complete`
