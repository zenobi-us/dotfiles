# Estimation Hierarchy (Miniproject-Aligned)

## Rules
- Idea: no numeric estimate.
- Epic: estimated in weeks/months.
- Story: estimated in story points (1,2,3,5,8,13).
- Task: estimated in story points (1,2,3,5,8).
- Task > 8 points SHOULD be split.

## Why
- Epic scale: roadmap/timeline.
- Story scale: planning/capacity.
- Task scale: execution predictability.
