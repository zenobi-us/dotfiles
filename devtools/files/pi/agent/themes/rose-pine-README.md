# Rosé Pine Theme

A beautiful, elegant dark theme inspired by the popular [Rosé Pine](https://rosepinetheme.com/) color scheme.

## Overview

Rosé Pine is a soho vibes color scheme for cozy, focused coding. This theme brings the distinctive purple-toned palette with carefully balanced accent colors to the Pi Coding Agent.

## Color Palette

### Base Colors
- **Base**: `#191724` - Deep purple-black background
- **Surface**: `#1f1d2e` - Slightly elevated surface
- **Overlay**: `#26233a` - Overlay elements and highlights

### Accent Colors
- **Foam**: `#9ccfd8` - Soft cyan (primary accent)
- **Pine**: `#31748f` - Muted teal (borders)
- **Iris**: `#c4a7e7` - Soft purple (keywords)
- **Rose**: `#ebbcba` - Warm rose (variables, inline code)
- **Gold**: `#f6c177` - Warm gold (strings, headings, warnings)
- **Love**: `#eb6f92` - Vibrant pink (errors, high thinking)

### Text Colors
- **Text**: `#e0def4` - Primary text
- **Subtle**: `#908caa` - Secondary/dimmed text
- **Muted**: `#6e6a86` - Tertiary/very dimmed text

## Features

- **Soho vibes aesthetic**: Cozy purple-toned backgrounds with warm accents
- **Excellent readability**: High contrast between text and backgrounds
- **Distinct syntax highlighting**: Each language element has its own personality
- **Thinking level progression**: Clear visual hierarchy from muted → foam → iris → love
- **Low eye strain**: Carefully balanced warm tones for long coding sessions

## Color Assignments

### UI Elements
- **Accent/Cursor**: Foam cyan
- **Borders**: Pine (normal), Foam (accent), Overlay (muted)
- **Success**: Foam
- **Error**: Love pink
- **Warning**: Gold

### Syntax Highlighting
- **Comments**: Muted gray
- **Keywords**: Iris purple
- **Functions**: Foam cyan
- **Variables**: Rose
- **Strings**: Gold
- **Numbers**: Rose
- **Types**: Foam cyan
- **Operators**: Text (default)
- **Punctuation**: Subtle

### Markdown
- **Headings**: Gold
- **Links**: Foam
- **Inline code**: Rose
- **Code blocks**: Foam
- **Quotes**: Subtle

### Tool Output
- **Pending**: Overlay background
- **Success**: Dark green-tinted (`#1f2827`)
- **Error**: Dark red-tinted (`#2a1f24`)
- **Diff added**: Foam
- **Diff removed**: Love

## Best For

- Developers who enjoy cozy, warm aesthetics
- Long coding sessions requiring low eye strain
- Purple theme enthusiasts
- Fans of the Rosé Pine design philosophy

## Usage

### Command Line
```bash
pi --theme rose-pine
```

### Interactive Selection
```bash
pi
/theme
# Select "rose-pine" from the list
```

### Settings File
```bash
pi
/settings
# Set "theme": "rose-pine"
# Save and restart
```

## Design Philosophy

Rosé Pine follows the principle of "soho vibes" - creating a comfortable, focused environment that feels like a cozy workspace. The purple-toned base with carefully selected warm accents creates a distinctive look that's both functional and aesthetically pleasing.

The theme maintains excellent readability while providing enough visual distinction between different syntax elements to aid comprehension without being overwhelming.
