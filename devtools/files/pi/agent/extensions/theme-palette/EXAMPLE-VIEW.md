### Contrast Validation
See the four-level hierarchy in real message bubbles:
- **Label** (bold accent) - User, AI, System
- **Primary text** - Message content
- **Secondary text** - Timestamps, metadata
- **Hints** - Placeholder text

### Real Usage Patterns

- Why `userMessageBg` is lighter (elevation)
- Why `dim` is used for disabled states
- How syntax colors create readable code blocks
- When to use `border` vs `overlay-1`

## Commands to Try


## Customization Ideas

Add your own UI simulations in `components/UISimulator.ts`:

1. **Navigation menus** - Show active/inactive states
2. **Table rows** - Demonstrate stripe patterns and hover
3. **Progress bars** - Show success/warning/error states
4. **Badges** - Count indicators, status labels
5. **Tooltips** - Overlay styling and contrast
6. **Dropdown menus** - Show focus and selection states
7. **Modal dialogs** - Demonstrate layering and shadows (via color)
8. **Breadcrumbs** - Show hierarchy with separators

Each simulation helps validate that your color choices work in practice!
