# Learning: Refactoring.Guru derivative authoring guardrails

## Summary
For this project, safe reuse means using Refactoring.Guru as a conceptual reference and taxonomy source while writing all instructional text from scratch.

## Details
- Content Usage Policy allows non-substantial text citation and limited illustration reuse with attribution.
- “All rights reserved” footer and premium eBook license constraints justify a conservative posture.
- Practical authoring mode: factual extraction + original prose + explicit references.

## Implications
- `dpat1102` template must include mandatory attribution fields and anti-copying rules.
- Future pattern skills should default to zero direct quotes and zero image reuse.
- Any exception path (long quotes / extra illustrations) should require explicit permission trace in the artifact.
